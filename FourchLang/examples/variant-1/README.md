## Variante 1 : jeu de "Snake" classique
Le code contenu dans le fichier `program.fl` de ce dossier représente la configuration d'un jeu de "Snake" classique, avec un serpent ayant un longueur de 2 cases, une grille de 10 cases de long par 9 cases de large, et un fruit déjà présent.<br>
Manger un fruit avec le serpent augmente le score de 1, et allonge le serpent d'une case. l'allongement n'est pas décrit dans le programme car implicite.<br>
Rentrer en contact avec une case ou se trouve le serpent met fin au jeu.
